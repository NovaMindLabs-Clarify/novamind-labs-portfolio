# ChairTime — Blackwood Barbershop Booking Platform

Premium-проект портфолио (см. `PRD.md`). React + TypeScript + Vite SPA.

## Важно: не открывается напрямую как `file://`

В отличие от тира 1 (`1_site`…`5_site`, чистый HTML/CSS/JS), это собранное React-приложение со сборкой через Vite. Собранный бандл использует `<script type="module">`, а браузеры блокируют загрузку ES-модулей по CORS-политике при открытии файла напрямую через `file://`. Поэтому нужен локальный HTTP-сервер:

```bash
npm install
npm run dev       # режим разработки, http://localhost:5173
# либо
npm run build      # продакшен-сборка в dist/
npm run preview    # раздаёт dist/ по HTTP, обычно http://localhost:4173
```

Роутинг — `HashRouter` (адреса вида `#/book`, `#/admin`), поэтому даже без настройки сервера на SPA-fallback все маршруты работают корректно.

## Демо-роли

На главном экране — переключатель на три роли (без реальной авторизации, это явно обозначено в интерфейсе):
- **Client** — `#/book` — флоу бронирования
- **Staff** — `#/staff` — дашборд барбера
- **Admin** — `#/admin` — доска зала, услуги, барберы, все записи

Данные — мок (см. `src/data/seed.ts`), хранятся в `localStorage`, сбрасываются очисткой хранилища браузера.
